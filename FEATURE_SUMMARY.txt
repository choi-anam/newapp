í¾¯ USAGE EXAMPLES:

Telegram Bot Setup:
  1. Log in to admin panel
  2. Go to Configuration > Telegram Bot
  3. Click "Add Configuration"
  4. Get token from @BotFather
  5. Fill in bot name and token
  6. Click "Save Configuration"

Email Setup (Gmail):
  1. Log in to admin panel
  2. Go to Configuration > Email Config
  3. Click "Add Configuration"
  4. Select "SMTP" as mailer type
  5. Set Host: smtp.gmail.com
  6. Set Port: 587
  7. Set Username: your-email@gmail.com
  8. Set Password: Your App Password (not regular password)
  9. Set Encryption: TLS
  10. Set From Address and Name
  11. Click "Save Configuration"
